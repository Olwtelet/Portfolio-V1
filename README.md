# 🚀 Overview

**Portfolio V1** is the first iteration of my personal portfolio website.

The project was created to present my professional background, highlight selected projects, and provide an easy way for recruiters, clients, and developers to learn more about my work.

The design follows a clean and minimalist approach, focusing on readability, performance, and user experience.

The portfolio includes:

- Modern minimalist interface
- Responsive design
- Experience timeline
- About section
- Featured projects

- Optimized performance

# ⚡ Quick Start

Clone the repository

```bash
git clone https://github.com/username/portfolio-v1.git
```

Open the project

```bash
cd portfolio-v1
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build production

```bash
npm run build
```
